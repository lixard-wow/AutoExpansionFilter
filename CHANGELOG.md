# Changelog

## Version 1.2.2

### Updated

- Updated for **World of Warcraft Patch 12.0.7**.
- Reviewed for **Patch 12.1.0** compatibility. No issues are expected with the update. Support for any new expansion content will be added as it becomes available.

## Version 1.2.1

### New

- Added support for **Current Expansion Only** in the **Crafting Orders** browser.
- Auction House and Crafting Orders can now be enabled or disabled independently.
- Added the addon version number to the settings window.

### Fixed

- Fixed the **Current Expansion Only** filter not staying enabled in the Crafting Orders browser.
- Fixed the Auction House filter checkbox not updating correctly after Blizzard UI changes.

## Version 1.0.0

- Initial release.
- Automatically enables the **Current Expansion Only** filter in the Auction House.
